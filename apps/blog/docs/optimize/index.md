# 性能优化

## 有哪些性能优化的手段？

https://segmentfault.com/a/1190000022205291

- 减少请求数：http2，雪碧图
- 降低体积：压缩
- 减少解析时间：服务端渲染
- 加快下载速度：CDN
- 预加载：preload
- 懒加载
- 按需加载
- 缓存
- 减少 dom 操作，减少重绘重排
- 任务分片
- 多线程：webworker
