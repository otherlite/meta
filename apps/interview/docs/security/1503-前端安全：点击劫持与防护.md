# 题目名
前端安全：点击劫持与防护

## 题目描述
什么是点击劫持（Clickjacking）？攻击者如何利用 iframe 覆盖诱导点击？如何在工程上防护？

## 题目答案
点击劫持：攻击者把受害站点页面嵌入到透明/半透明 iframe 中，在上层布置诱导按钮，诱导用户点击从而在受害站点执行敏感操作。

防护手段：
- `X-Frame-Options` 响应头：
  - `DENY`：禁止任何站点嵌入
  - `SAMEORIGIN`：仅同源可嵌入
- CSP `frame-ancestors`：更现代、更灵活的控制方式，可指定允许嵌入的源。
- 对关键操作增加二次确认/校验，降低误触带来的风险。

面试加分点：
- `X-Frame-Options` 与 CSP 的作用层级：两者都能防 iframe 嵌入，但 CSP 更推荐。
