# 题目名
判断数组与判断对象的多种方式

## 题目描述
如何判断一个值是否为数组/普通对象？`typeof`、`instanceof`、`Array.isArray`、`Object.prototype.toString` 各自有什么限制？

## 题目答案
- 数组：优先 `Array.isArray(x)`（跨 iframe 更可靠）。
- `instanceof Array`：受原型链与跨 realm 影响。
- 普通对象：常用 `Object.prototype.toString.call(x) === '[object Object]'`，或判断原型链。
- `typeof`：只能区分原始类型与 function，`typeof null === 'object'` 是历史坑。
