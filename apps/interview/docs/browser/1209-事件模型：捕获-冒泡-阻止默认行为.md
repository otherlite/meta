# 题目名
事件模型：捕获、冒泡与阻止默认行为

## 题目描述
请说明 DOM 事件的捕获与冒泡阶段，`addEventListener` 的第三个参数有什么作用？`preventDefault` 与 `stopPropagation` 分别影响什么？

## 题目答案
事件流（简化）：
- 捕获阶段：从 window/document 向下到目标节点。
- 目标阶段：到达目标节点。
- 冒泡阶段：从目标节点向上回到 window/document。

`addEventListener(type, listener, options)`：
- `capture: true` 表示在捕获阶段触发监听器。
- 默认是冒泡阶段触发。

`preventDefault()`：
- 阻止默认行为（如链接跳转、表单提交、滚动等），不影响事件传播。

`stopPropagation()`：
- 阻止事件继续传播（捕获/冒泡链路），不一定阻止默认行为。

`stopImmediatePropagation()`：
- 除了阻止传播，还会阻止当前元素上同类型后续监听器执行。
