# 题目名
HTTP 请求头与响应头常见字段

## 题目描述
请列举并解释常见请求头与响应头的用途：`Accept`、`Content-Type`、`Authorization`、`Cookie`、`Cache-Control`、`ETag`、`Vary` 等。

## 题目答案
- 请求头：
  - `Accept/Accept-Encoding` 声明可接受的类型/压缩。
  - `Content-Type` 描述请求体格式。
  - `Authorization` 鉴权信息。
- 响应头：
  - `Cache-Control/ETag` 缓存策略。
  - `Vary` 告诉缓存按哪些请求头区分内容（避免错误复用）。
