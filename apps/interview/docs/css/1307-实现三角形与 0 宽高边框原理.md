# 题目名
用 CSS 实现三角形与“0 宽高边框”原理

## 题目描述
为什么设置 `width/height: 0` 再配合 `border` 可以画出三角形？如何画不同方向的三角形？工程中有哪些更推荐的替代方案？

## 题目答案
原理：
- 当元素宽高为 0 时，边框四个方向会以斜边方式相交。
- 通过把三个方向的 `border-color` 设为透明，仅保留一个方向的颜色，就形成一个三角形。

方向控制：
- 想要向上的三角形：保留下边框颜色，其余透明。
- 向下：保留上边框颜色。
- 向左/向右同理。

工程替代：
- 图标类优先用 SVG（清晰、可控、易动画）。
- 复杂形状可用 SVG/clip-path（注意兼容）。
